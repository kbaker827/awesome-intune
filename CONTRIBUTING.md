# Contributing to Awesome Intune

Thank you for your interest in contributing to this list!

## How to Contribute

1. **Fork** this repository
2. **Create a branch** for your changes
3. **Add your resource** following the format below
4. **Submit a pull request**

## Format Guidelines

When adding a new resource, please include:

```markdown
- [Resource Name](https://github.com/username/repo) ⭐ X - Brief description of what it does and why it's useful.
```

Example:
```markdown
- [DeviceOffboardingManager](https://github.com/ugurkocde/DeviceOffboardingManager) ⭐ 226 - PowerShell GUI tool for managing and offboarding devices from Intune, Autopilot, and Entra ID.
```

## Categories

Please add resources to the most appropriate category:
- Device Management
- Autopilot
- macOS Management
- Win32 App Packaging
- Configuration Profiles
- PowerShell & Automation

## Quality Standards

- Resources should be actively maintained (updated within last 2 years)
- Include a clear description
- Must be related to Microsoft Intune
- Prefer open-source tools and scripts

## Star Counts

Star counts are helpful but optional. Format: `⭐ XXX`

Update star counts periodically if maintaining this list.

## Code of Conduct

- Be respectful and constructive
- Focus on quality over quantity
- Help others learn and improve

Thank you for helping make this list more awesome!
